NectarView 是一款适用于 macOS 的快速、轻量级漫画和图片浏览器——现已直接支持 RAR、7z、ZIP、TAR 及 30 余种压缩格式，无需解压。灵感来源于 Windows 上 HoneyView 的易用性，充分发挥 macOS 原生特性，带来流畅的阅读体验。

直观的操作方式
	•	支持拖放方式轻松打开文件
	•	通过右键菜单打开文件
	•	双击切换全屏模式
	•	拖动浏览区域以移动窗口

多格式压缩包支持
	•	支持 RAR、7z、ZIP、TAR、LhA、CAB、StuffIt 等多种格式
	•	支持加密压缩包（密码保护）
	•	根据文件内容自动识别格式

灵活的浏览选项
	•	可在单页与双页展开视图之间切换
	•	支持从左到右和从右到左的阅读方式
	•	通过滑动条快速浏览图片
	•	双页模式下呈现逼真的书本外观

高性能表现
	•	无需解压，直接打开压缩包
	•	图片预加载，显示更迅速
	•	充分利用 macOS 原生 API 高效处理

友好的用户界面
	•	简洁、无干扰的界面设计
	•	可自定义背景及控制栏颜色
	•	macOS 原生风格设置窗口
	•	支持键盘快捷键操作

多语言支持
	•	提供英文和日文界面
	•	根据系统语言自动切换

高级功能
	•	书签功能，保存喜爱的阅读位置
	•	支持批量显示文件夹内的图片
	•	支持 PDF 文件，Retina 高清渲染
	•	自动翻页，可调节间隔时间
	•	图片旋转及滤镜效果（复古、单色、黑白等）
