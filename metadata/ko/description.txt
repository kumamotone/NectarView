NectarView는 macOS용 빠르고 가벼운 만화 및 이미지 뷰어입니다. RAR, 7z, ZIP, TAR 등 30가지 이상의 압축 형식을 압축 해제 없이 바로 열 수 있습니다. Windows의 HoneyView에서 영감을 받아 macOS 네이티브 기능을 활용하여 부드러운 읽기 경험을 제공합니다.

직관적인 조작
	•	드래그 앤 드롭으로 파일을 쉽게 열기
	•	컨텍스트 메뉴에서 파일 열기
	•	더블 클릭으로 전체 화면 모드 전환
	•	보기 영역을 드래그하여 창 이동

다중 압축 형식 지원
	•	RAR, 7z, ZIP, TAR, LhA, CAB, StuffIt 등 지원
	•	비밀번호로 보호된 압축 파일 지원
	•	파일 내용에 따른 자동 형식 감지

유연한 보기 옵션
	•	단일 페이지와 양면 펼침 보기 전환
	•	좌에서 우, 우에서 좌 읽기 방향 지원
	•	슬라이더로 이미지를 빠르게 탐색
	•	펼침 보기에서 실감나는 책 모양 표현

고성능
	•	압축 해제 없이 압축 파일을 바로 열기
	•	이미지 프리페치로 빠른 표시
	•	macOS 네이티브 API를 활용한 효율적인 처리

사용자 친화적 인터페이스
	•	간결하고 방해 없는 UI
	•	배경 및 컨트롤 바 색상 사용자 지정 가능
	•	macOS 네이티브 설정 창
	•	키보드 단축키 지원

다국어 지원
	•	영어 및 일본어 인터페이스 제공
	•	시스템 언어에 따라 자동 전환

고급 기능
	•	북마크 기능으로 좋아하는 위치 저장
	•	폴더 내 이미지 일괄 표시
	•	Retina 렌더링으로 PDF 파일 지원
	•	조절 가능한 간격의 자동 페이지 넘기기
	•	이미지 회전 및 필터 효과 (세피아, 모노, 느와르 등)
