NectarView 是一款快速、輕量的 macOS 漫畫與圖片瀏覽器——直接支援 RAR、7z、ZIP、TAR 及 30 種以上壓縮格式，無需解壓縮。靈感來自 Windows 上 HoneyView 的操作體驗，充分運用 macOS 原生功能，帶來流暢的閱讀享受。

直覺操作
	•	透過拖放輕鬆開啟檔案
	•	從右鍵選單開啟檔案
	•	雙擊切換全螢幕模式
	•	拖曳瀏覽區域移動視窗

多格式壓縮檔支援
	•	支援 RAR、7z、ZIP、TAR、LhA、CAB、StuffIt 等格式
	•	支援密碼保護的壓縮檔
	•	依檔案內容自動偵測格式

彈性瀏覽選項
	•	切換單頁或雙頁跨頁顯示
	•	支援從左到右及從右到左閱讀方向
	•	透過滑桿快速瀏覽圖片
	•	跨頁模式呈現逼真的書本外觀

高效能表現
	•	直接開啟壓縮檔，無需解壓縮
	•	圖片預載入，顯示快速流暢
	•	運用 macOS 原生 API 進行高效處理

友善使用介面
	•	簡潔、不干擾的操作介面
	•	可自訂背景與控制列顏色
	•	macOS 原生設定視窗
	•	支援鍵盤快捷鍵操作

多語言支援
	•	提供英文與日文介面
	•	依系統語言自動切換

進階功能
	•	書籤功能，儲存喜愛的位置
	•	批次顯示資料夾中的圖片
	•	支援 PDF 檔案並以 Retina 解析度渲染
	•	可調整間隔的自動翻頁功能
	•	圖片旋轉與濾鏡效果（懷舊、單色、黑色電影等）
